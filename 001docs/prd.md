# 产品需求文档（PRD）

## 1. 产品概述
Friday 是一款科研向的本地 AI 助手，负责将 PDF、视频、音频等非结构化资料转化为结构化知识并支持进一步检索、问答与分析。

## 2. 目标用户
- 科研人员 / 研究生
- 技术从业者
- 需要高频处理文献、课程视频、会议录音的用户

## 3. 核心价值
1. 非结构化 → 结构化（PDF/视频/音频 → Markdown）
2. 自动化处理流程（下载 / 解析 / 总结 / 导出）
3. 全本地隐私与高性能（Tauri + Python）
4. 可扩展的插件化系统

## 4. 功能范围（MVP）
- **PDF 解析**：文本提取、公式、图表、Markdown 导出
- **视频总结**：字幕抽取、长文本总结、关键片段
- **音频转写**：ASR、说话人分离、逐字稿导出
- **Command 模式**：自然语言指令触发任务

## 5. 不做的范围
- 不做通用大模型聊天
- 不做代码 IDE
- 不做云端服务

## 6. 用户故事
### US-01 论文处理
用户拖入一篇论文 → Friday 输出结构化 Markdown → 用户提问并分析内容。

### US-02 视频学习
用户输入课程视频链接 → Friday 下载字幕 → 生成总结与关键章节。

### US-03 会议转写
用户导入录音文件 → Friday 自动转写并区分发言人 → 导出文本。

## 7. 功能需求（摘要）
- 提供统一资源库（PDF/Video/Audio）
- 提供可编辑 Markdown 视图
- 提供 AI 问答入口
- 支持插件自动注册

## 8. 业务流程简图
1. 用户输入文件/链接
2. Friday 判断类型 → 调用对应模块
3. Python Sidecar 执行处理
4. 返回结构化结果
5. 用户可继续问答 / 导出
