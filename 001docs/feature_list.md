# Friday 功能清单（Feature List）

## A. PDF 模块（Friday-Reader）
1. PDF 文件导入
2. 版面分析（单栏/双栏）
3. 文本抽取
4. 数学公式识别（LaTeX 输出）
5. 图表提取并保存
6. Markdown 渲染与实时预览
7. Markdown 导出
8. 资源存储到 Library

## B. 视频模块（Friday-Watcher）
1. 视频链接解析（YouTube/Bilibili）
2. 视频元数据获取（标题/封面/时长）
3. 字幕获取（自动/内置）
4. 全文总结（长上下文大模型）
5. 时间轴关键片段检测
6. 内容问答（基于视频字幕）
7. 结果存储到 Library

## C. 音频模块（Friday-Listener）
1. 多格式音频导入
2. 音频波形展示
3. ASR 转写（Faster-Whisper）
4. 说话人分离（Pyannote）
5. 术语校正（提示词）
6. 逐字稿导出（TXT/MD）

## D. AI Agent（Friday-Core）
1. 自然语言任务指令
2. 意图识别（PDF/Video/Audio）
3. Task Queue 模型
4. Friday 人格化回复
5. 任务日志与状态显示

## E. 系统功能
1. Library 页面管理资源
2. 统一资源模型（Resource Schema）
3. 插件系统（plugins/ 文件夹自动识别）
4. 设置页（API Key、安全性、本地路径）
5. 日志系统（本地记录）
